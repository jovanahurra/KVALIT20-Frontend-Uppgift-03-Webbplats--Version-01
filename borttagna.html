Borttagna element och attribut

Borttagna element

Eftersom det bortfaller en del element kommer de att med tiden ersättas av andra element, eller av andra webbtekniker. För en lista över bortfallna element och hur de bör ersättas, se här.
Element som är till för presentation kommer att tas bort och kan istället ersättas med CSS. Dessa är:

  • BASEFONT – Sätter base font.
  • BIG – Renderar text i en större font.
  • CENTER – Centrerar innehåll.
  • FONT – Sätter font.
  • STRIKE – Drar ett streck genom texten.
  • TT – Renderar text i en monospace font (eller s.k. teletype font).
  • U – Drar ett streck under texten.

Elementen frame, frameset och noframes är borttagna också.

Övriga borttagna element inkluderar:

  • ACRONYM - Definierar innehåll att vara akronym. Ersätts av elementet abbr.
  • APPLET - Inkluderar en s.k. applet på hemsidan. Ersätts av elementet object.
  • ISINDEX - Indikerar att sidan är ett index dokument. Ersätts av formulär.
  • DIR - Definierar en s.k. directory list. Ersätts av elementet ul.

Borttagna attribut

Även en del attribut försvinner. Här kommer vi endast presentera de största ändringarna.

  • Attributet profile tas bort från HEAD.
  • Attributet version tas bort från HTML.
  • Attributet name tas bort från IMG och ersätts av attributet id.
  • Attributen valuetype och value tas bort från PARAM.
  • Attributet align kommer tas bort från alla strukturella element, såsom TR, TABLE, DIV, HR, etc. Istället ersätts det av CSS.
  • Attributet background och bgcolor tas bort från BODY.
  • Attributen border, ellpadding, cellspacing, frame tas bort från TABLE.
  • Attributen height och height tas bort från TH och TD.
